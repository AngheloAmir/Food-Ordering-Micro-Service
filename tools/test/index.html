                                1
                                Introduction
                            
    
                            
    
                                
    
                                    Welcome to the API Tester Tool. This application is designed to streamline the development and testing lifecycle of your microservices. It combines manual endpoint testing, automated regression suites, and database seeding into a single, cohesive interface.
                                
    
                                
    
                                    The tool is fully client-side (HTML/JS) but interacts directly with your running backend services. It persists your preferences (like Base URL and Notes) to your browser's Local Storage.

                                3
                                Key Features
                            
    
                            
    
                                
                                
    
                                    
    
                                        
                                    
    
                                    
    
                                        
    Authentication
    
                                        
    
                                            Quickly switch between user roles (Admin, Customer, Vendor). Login tokens are automatically saved to localStorage and attached to subsequent requests.
                                        
    
                                    
    
                                
    

                                
                                
    
                                    
    
                                        
                                    
    
                                    
    
                                        
    Request Builder
    
                                        
    
                                            Navigate endpoints via the sidebar. Use Presets to quickly populate bodies. Responses are syntax-highlighted for readability. Real-time stopwatch and dynamic URL updates included.
                                        
    
                                    
    
                                
    

                                 
                                 
    
                                    
    
                                        
                                    
    
                                    
    
                                        
    Automated Testing
    
                                        
    
                                            Define test cases in test.js. Run them all with one click. Failures are logged with a detailed diff view (Output vs. Expected).
                                        
    
                                    
    
                                
    

                                 
                                 
    
                                    
    
                                        
                                    
    
                                    
    
                                        
    Seeding
    
                                        
    
                                            Populate your local database with initial data. Supports batch "Seed All" execution with progress tracking.
